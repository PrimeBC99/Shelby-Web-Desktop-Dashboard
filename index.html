Erasure Coding
File dipecah menjadi k fragmen data + m fragmen parity menggunakan algoritma Reed-Solomon. Data bisa dipulihkan selama minimal k dari k+m fragmen tersedia — bahkan jika beberapa node offline atau corrupt.
Contoh: file.jpg → 4 data + 2 parity (toleransi 2 node gagal)

Audit Protocol
Smart contract on Aptos secara berkala mengirim challenge acak ke node. Node harus membuktikan mereka benar-benar menyimpan data dengan mengirim cryptographic proof. Node yang gagal audit akan dipotong reward-nya.
// Audit cycle — setiap epoch
challenge = randomBytes(32)
proof = node.computeProof(challenge)
valid = contract.verify(proof) ✓
